# 🎬 CineVerse

CineVerse is a full-stack movie discovery and booking platform inspired by Netflix and IMDb.

## 🚀 Features

- User Authentication
- Movie Catalog
- Protected Routes
- Dashboard UI
- Booking Interface
- React Router Navigation
- JWT-based Authentication (UI Level)
- Responsive UI Design

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router
- CSS3
- Axios

### Backend (Planned)
- Spring Boot
- PostgreSQL
- MongoDB
- Redis
- RabbitMQ
